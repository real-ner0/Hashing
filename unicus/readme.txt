unicus : means "only" since hashes are in general unique.

unicus is a simple ruby based script, just fed the the text input in the command line, hit enter and it'll show the hash in MD5, SHA1,256 and 512.

paste the .rb file in /bin.

root@kali:~# unicus abc

MD5:
